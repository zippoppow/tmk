## üîç What's Under the Hood

### Algorithm
1. Fetches morpheme database from TMK API (~1000+ morphemes)
2. Sorts morphemes by length (longest first)
3. For each word, matches morphemes left-to-right
4. Uses greedy approach for best matches
5. Detects prefix/suffix types and adds notation
6. Returns complete breakdown with metadata

### Technology
- **Frontend:** React, Material-UI, Next.js
- **Backend:** Node.js, Next.js API routes
- **Database:** TMK API (MongoDB)
- **Styling:** Emotion, Material-UI sx prop
- **Data Format:** JSON/CSV

### Performance
- Morpheme database load: ~500ms
- Per-word processing: 5-10ms
- 100 words: 1-2 seconds
- Batch size: Up to 1000 words
