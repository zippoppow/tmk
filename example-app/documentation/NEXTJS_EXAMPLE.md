# Pegasus NextJS Example

NextJS app consuming the Pegasus Component Library.

## Quick Start

```bash
cd example-app
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to see the example.

## Integration

This example demonstrates importing:
- **Tokens** from `../tokens/global.json`
- **Theme components** from `../theme/index.js` (Button, Card, Badge, etc.)
- **Icons** from `../icons/filled/` and `../icons/line/`

See [example-app/README.md](./example-app/README.md) for detailed setup and patterns.
